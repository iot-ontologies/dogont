Simple Device Modeling using existing classes

Device modeling (instantiation) in DogOnt involves 7 steps:

1 identify the object to represent and the corresponding DogOnt concept (if exists)

2 model the object according to the DogOnt class definition including functionalities and states

3 define individuals for additional functionalities and states

4 model any object that is functional to the correct representation of the initial object

5 model the environment(s) in which the objects are placed

6 model any explicit relation between instances

7 model the network-specific information allowing to interface real-devices
